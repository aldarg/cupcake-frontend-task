# Testing task for Cupcake Development

React frontend task:

- React with hooks
- Typescript
- Webpack + Babel

## Deployment

```bash
$ docker-compose up
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
#build
$ yarn build

#dev server
$ yarn dev

#production mode
$ yarn start:prod
```
